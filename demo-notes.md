# Demo prompts

- Why does chunk overlap improve retrieval quality?
- What metrics should an AI API gateway track?
- How should model evaluations be made reproducible?
- Why can large chunks reduce precision?
- What should inference benchmarks preserve for debugging?

# Demo workflow

1. Open the app.
2. Review the seeded documents and chunk preview.
3. Generate embeddings.
4. Run one of the demo prompts.
5. Compare the strategy table to see how chunk size changes top score and chunk count.
6. Switch chunking mode and preview new chunks.
